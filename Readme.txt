Описание программно-аппаратной системы, состоящей из сервера и сети устройств.

Для запуска системы нужно проделать следующие шаги: server.py, proxy.py, device.py 

В директории test_task имеется 3 файла: : proxy.py, server.py, device.py 

Для запуска системы нужно проделать следующие шаги:

1) Запустить из командной строки файл proxy.py:

	"python proxy.py"

Прокси сервер, по умолчанию, запускается по ip адресу: "127.0.0.1" с номером порта "8889"

2) Запустить из командной строки файл server.py:

	"python server.py (номер порта)"

Ниже приведена таблица соответствия прокси сервера, которая находится в директории "". 
Целесообразно запускать сервера по содержащимся в таблице адресам, а устройства - c содержащимися в таблице id.
Таблица соответствия прокси сервера находится в следующей директории: "test_task/proxy/table.xml"
_____________________________________
     staging 127.0.0.1:8888

     ID1: 1
     ID2: 2
     ID3: 3
		
     production_like 127.0.0.1:8887

     ID1: 4

     ID3: 6
     ID4: 7
		
     production value 127.0.0.1:8885

     ID1: 8

Так же, согласно заданию, имеется возможность запуска сервера с номером порта: 8886, на который будут приходить запросы устройств, 
id которых не присутствует в таблице.
Есть возможность установить время задержки ответа на запрос. Это можно сделать в "test_task/server/config.xml"
Серверы, по умолчанию, запускаются по адресу "127.0.0.1". Не следует запускать сервер с номером порта "8889".


3) Запустить из командной строки файл device.py:

	"python device.py 127.0.0.1 8889 (id устройства)"

После запуска устройство начнёт посылать запросы на прокси сервер. 
Согласно заданию, устройство должно посылать запрос раз в 5 минут. Для наглядности работы системы, временной промежуток был уменьшен до нескольких секунд. 
Его можно изменить в конфигурационном файле: "test_task/device/delay.xml".

В процессе работы системы будут формироваться и заполняться log-файлы. 

- Для устройств это будут: "test_task/device/(id устройства)_device.xml".
- Для прокси это будет: "test_task/proxy/proxy_info.xml".
- Для серверов это будут: "test_task/proxy/(номер порта)_server.xml"
